// Description
	Collaboration is a program for creating tools for C++ game development. This means
	that the program makes different utilities for c++ classes, functions and modules. The main focus
	is game development. 

// Steps for Collaboration

1. What tool are we going to create for c++?
2. Generate it with Generator Tool Engine.
3. Test it.
4. Use it.

// List of Game engine parts for creating tools

1. Render Engine
2. Physics Engine
3. Collision Detection
4. Sound Engine
5. Scripting
